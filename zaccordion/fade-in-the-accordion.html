zAccordion — Fade in the Accordion

One of the drawbacks when it comes to JavaScript slideshows and other post-render solutions is the split second just before the plugin build is complete. Depending on the page's CSS, for a brief moment, images may appear to be stacked one after another or even render on top of each other.

In the example below, a function is called that will fade in the accordion once it is ready.

This example was not built to support smaller screen sizes such as mobile devices. However, the latest version of zAccordion can be configured to support these devices using media queries. Please see the example demonstrating responsive design.

Implementation

  • HTML
  • JavaScript
  • CSS

  • I've included two wrapper divs. The first div (outer) will be set roughly to the height of the final accordion. It will serve as a placeholder. The second div (inner) will initially be hidden from view.

                          <div id="outer">
                        <div id="inner">
                          <ul id="slides">
                            <li>
                              <img src="bottles.jpg" alt="" />
                            </li>
                            <li>
                              <img src="barrels.jpg" alt="" />
                            </li>
                            <li>
                              <img src="rack.jpg" alt="" />
                            </li>
                            <li>
                              <img src="cellar.jpg" alt="" />
                            </li>
                          </ul>
                        </div>
                      </div>

  • The buildComplete option is set to a function that will fade in the inner div once the accordion is ready.

                          $(window).load(function() {
                        $("#slides").zAccordion({
                          width: 700,
                          slideWidth: 400,
                          height: 400,
                          buildComplete: function () {
                            $('#inner').fadeIn(3000);
                          }
                        });
                      });

  • Styling the wrapper divs is easy.

                          #outer{height:400px;margin:0 auto;width:700px;}
                      #inner{display:none;position:relative;}
