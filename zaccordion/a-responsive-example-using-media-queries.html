zAccordion — A Responsive Example using Media Queries

Using a JavaScript library such as Enquire.js, zAccordion can be built to respond to media queries. The example below shows how this is accomplished.

Implementation

For implementation instructions, please download zAccordion.

  • HTML
  • JavaScript
  • CSS

  • The Enquire.js file is included along with jQuery and zAccordion.

                            <script src="enquire.min.js"></script>

    The HTML is very simple. This example uses an unordered list and images.

                            <ul id="media">
                          <li>
                            <img src="sign.jpg" alt="" />
                          </li>
                          <li>
                            <img src="graffiti.jpg" alt="" />
                          </li>
                          <li>
                            <img src="dragon.jpg" alt="" />
                          </li>
                          <li>
                            <img src="bamboo.jpg" alt="" />
                          </li>
                        </ul>

  • The selector will be stored for later use.

                            var media = $('#media');

    A JavaScript object is created to hold the default options.

                            var defaults = {
                          easing: 'easeOutExpo',
                          slideClass: 'media',
                          buildComplete: function () {
                            media.css('visibility', 'visible');
                          },
                          timeout: 5500
                        };

    A function named build() will create the initial accordion. It will accept a JavaScript object containing new values for slideWidth, width, and a height. Each time the function is called, the current accordion is destroyed and rebuilt with the new options.

                            function build(x) {
                          var opts, current;
                          if (!$.isEmptyObject(media.data())) { /* If an zAccordion is found, rebuild it with new settings. */
                            media.css('visibility', 'hidden');
                            current = media.data('current');
                            opts = $.extend({
                              startingSlide: current
                            }, defaults, x);
                            media.zAccordion('destroy', {
                              removeStyleAttr: true,
                              removeClasses: true,
                              destroyComplete: {
                                afterDestroy: function() {
                                  try {
                                    console.log('zAccordion destroyed! Attempting to rebuild...');
                                  } catch (e) {}
                                },
                                rebuild: opts
                              }
                            });
                          } else { /* If no zAccordion is found, build one from scratch. */
                            media.css('visibility', 'hidden');
                            opts = $.extend(defaults, x);
                            media.zAccordion(opts);
                          }
                        }

    Now the Enquire.js library comes in handy. The library will help listen for media queries and fire the build() function when it finds a match.

                            /* A unique Media Query is registered for each screen size. */
                        enquire.register('screen and (min-width: 960px)', { /* Standard screen sizes and a default build for browsers that are unsupported. */
                          match : function () {
                            build({
                              slideWidth: 600,
                              width: 900,
                              height: 345
                            });
                          } /* The *true* value below means this media query will fire by default. */
                        }, true).register('screen and (min-width: 768px) and (max-width: 959px)', {
                          match : function () {
                            build({
                              slideWidth: 460,
                              width: 700,
                              height: 265
                            });
                          }
                        }).register('screen and (min-width: 480px) and (max-width: 767px)', {
                          match : function () {
                            build({
                              slideWidth: 250,
                              width: 380,
                              height: 145
                            });
                          }
                        }).register('screen and (max-width: 479px)', {
                          match : function () {
                            if (!$.isEmptyObject(media.data())) {
                              media.zAccordion('destroy', {
                                removeStyleAttr: true,
                                removeClasses: true,
                                destroyComplete: {
                                  afterDestroy: function() {
                                    try {
                                      console.log('zAccordion destroyed!');
                                    } catch (e) {}
                                  }
                                }
                              });
                            }
                          }
                        }).listen(5);

    In this example, mobile device sizes (the last media query above) show zAccordion's slides stacked on top of each other.

  • Images sizes are set with a max-width property. Another method is to swap out images with JavaScript, possibly using an alternate source set in a data-src attribute. There is also some default styling that will clean up the display for smaller screen sizes.

                            #media {list-style:none;visibility:hidden;}
                        #media img {display:block;max-width:100%;}
                        #media li {width:100%;}

Notes Regarding Responsive Design

This website's layout responds to CSS media queries. However, zAccordion depends on JavaScript, and requires some way in JavaScript to detect when a media query is matched. The example above achieves this using Enquire.js.

An alternative to the method above is to build zAccordion using percentages instead of pixels.
